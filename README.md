# 📂 Terminal File Manager (Python)

A simple Terminal File Manager built with Python using the `os` module. This project is created for learning Python by implementing real file management operations.

## ✨ Features

- 📍 Show Current Directory
- 📂 Change Directory
- 📋 List Files & Folders
- 🔍 Search File
- 📄 Create File
- 📁 Create Folder
- ✏️ Rename File/Folder
- 🗑️ Delete File
- 🗂️ Delete Folder
- ℹ️ Show File Information

## 🛠️ Technologies Used

- Python 3
- os Module
- time Module
- Exception Handling
- Functions
